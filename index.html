Agent Instructions

  1. Start with brand-kit.json; it is canonical and machine-readable.
  2. Primary color is Navy #00365B; primary typeface is Gibson (free alternative: Montserrat).
  3. The logo is black or white only; never recolor it.
  4. On the HoustonsFirstV1 web theme, use theme classes; never hard-code hex in markup.
  5. Run the checklist array in brand-kit.json before finalizing any asset.
